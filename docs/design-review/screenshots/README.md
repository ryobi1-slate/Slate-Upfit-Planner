# Screenshot Records

Use this naming pattern:

`YYYY-MM-DD_viewport_page_state_description.png`

Example:

`2026-07-21_390x844_staging_empty-state.png`

Screenshot rules:

- Capture at 100% browser zoom.
- Record the viewport size.
- Avoid customer or dealer data.
- Do not commit screenshots by default.
- Add screenshots only when needed for a reviewed PR or defect record.
- Confirm repository policy before committing binary files.
