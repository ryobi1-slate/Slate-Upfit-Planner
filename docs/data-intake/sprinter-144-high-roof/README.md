# Mercedes-Benz Sprinter 144 High Roof Intake

This package is the controlled intake workspace for proposed 144 High Roof vehicle
records. Published configuration facts remain draft intake evidence; this directory
contains no approved vehicle record or production geometry.

Complete `source-index.csv` first, then record measurements and applicability in the related worksheets. Use `engineering-verification.md` and `data-approval-signoff.md` only after the evidence package is complete.

Photos support measurements but do not replace source documents or calibrated worksheets.

> **Payload warning:** Do not reuse 4211 lb as a general Sprinter 144 High Roof payload value.

RWD is the primary commercial baseline. AWD remains supported as a separate payload
configuration. RWD and AWD geometry sharing is only a candidate pending explicit source
or physical verification.

See the repository-level 2026 configuration matrix and source-exception register before
using any SRC-004 value. Every payload row remains configuration-specific and requires
final VIN verification.

PR B remains blocked until all blocking fields are known, sources are indexed, engineering verification is complete, open exceptions are resolved or accepted, and the package is approved.
