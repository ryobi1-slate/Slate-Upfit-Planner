# Mercedes-Benz Sprinter 170 Extended High Roof Intake

This directory reserves a documentation-only intake package for 170 Extended High Roof
body geometry. It contains no production geometry or production-selectable vehicle data.

Use the shared intake templates under `docs/data-intake/` and the 2026 configuration
matrix for identity and payload evidence. Do not infer the absent 170 Extended 3500XD
RWD SRC-004 configuration. RWD and AWD geometry sharing remains a candidate only;
payload remains configuration-specific.
