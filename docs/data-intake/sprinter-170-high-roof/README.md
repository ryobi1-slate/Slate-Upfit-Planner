# Mercedes-Benz Sprinter 170 High Roof Intake

This directory reserves a documentation-only intake package for 170 High Roof body
geometry. It contains no production geometry or production-selectable vehicle data.

Use the shared intake templates under `docs/data-intake/` and the 2026 configuration
matrix for identity and payload evidence. RWD and AWD may share a candidate geometry
group only after explicit source or physical verification. Payload remains
configuration-specific.
